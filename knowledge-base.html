TARS 20 Questions - Knowledge Base

Overview

TARS 20 Questions is an interactive 20 Questions game where you can play against an AI. The game features two modes:

  • AI Guesses: You think of an object, and the AI tries to guess it by asking you yes/no questions.
  • You Guess: The AI chooses an object, and you try to guess what it is by asking yes/no questions.

Architecture

The system is built with a modern web stack, utilizing a lightweight frontend and an AI-powered backend for gameplay logic.

Runtime

The application runs on a scalable infrastructure. The client is a web application accessible via modern browsers, while the AI backend is deployed on a cloud platform (e.g., Cloudflare Workers) to ensure low latency and high availability.

Features

  • Two interactive game modes.
  • Real-time AI responses.
  • Responsive, dark-themed UI with cyan accents.
  • Progressive Web App (PWA) capabilities.

Commands

Useful commands for development and maintenance:

  • npm run dev - Start the local development server.
  • npm run build - Build the client for production.
  • npm run deploy - Deploy the application to the hosting environment.
  • npm run test - Execute the test suite.
